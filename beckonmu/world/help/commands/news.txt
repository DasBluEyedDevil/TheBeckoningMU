|cCOMMAND:|n
  |wnews|n
  |w+news|n

|cDESCRIPTION:|n
  Read game news, announcements, policies, and important information.
  News is organized into categories for easy navigation.

|cUSAGE:|n
  |wnews|n                         - List all news categories
  |wnews <category>|n              - List topics in a category
  |wnews <category>/<topic>|n      - Read a specific news entry

|cEXAMPLES:|n
  |wnews|n                         - Show available categories
  |wnews general|n                 - List general news topics
  |wnews general/welcome|n         - Read welcome message
  |wnews updates|n                 - List game updates
  |wnews updates/changelog|n       - Read changelog
  |wnews policy|n                  - List game policies
  |wnews policy/rules|n            - Read game rules

|cCATEGORIES:|n
  |wgeneral|n  - Welcome messages, theme info, getting started
  |wupdates|n  - Game updates, changes, new features
  |wpolicy|n   - Rules, policies, and guidelines

|cTIPS:|n
  • Check |wnews updates|n regularly for new features and changes
  • Read |wnews policy/rules|n to understand game expectations
  • New players should start with |wnews general/getting_started|n
  • News files support ANSI colors and formatting
