|wTheBeckoningMU|n is a text-based roleplaying game set in the World of
Darkness, using the Vampire: The Masquerade 5th Edition rules. This
game focuses on the eternal struggle between vampires and their Beast,
the political machinations of the Camarilla, and the personal horror
of unlife in the modern nights.

|W▸ GETTING STARTED|n

If you're new to the game, try these commands:

  |y→|n |Whelp|n                - Access the help system
  |y→|n |W+news updates|n        - Read about recent changes
  |y→|n |W+news policy|n         - Read game policies
  |y→|n |W+bbs|n                 - Read the bulletin boards
  |y→|n |W+who|n                 - See who's online

For character creation, use the in-game chargen system:
  |y→|n |W+chargen/start|n       - Begin character creation

Or visit the web interface at:
  |yhttp://localhost:4001/character-creation/|n
