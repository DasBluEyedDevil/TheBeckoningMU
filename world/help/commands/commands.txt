|W                        COMMAND REFERENCE                           |n
|[R═══════════════════════════════════════════════════════════════════|n

TheBeckoningMU commands are organized by category. Use |yhelp <command>|n
for detailed information on any specific command.

|WBASIC COMMANDS:|n

|ylook|n               - Examine your surroundings or an object
|yinventory|n (|yi|n)     - View your inventory
|ysay <text>|n         - Speak out loud (visible to room)
|ypose <action>|n      - Describe an action or emote
|ywhisper <who>=<msg>|n - Speak privately to someone
|ywho|n                - See who is online
|yhelp <topic>|n       - View help on a topic

|WV5 GAME MECHANICS:|n

|y+sheet|n             - View your character sheet
|y+roll <pool>|n       - Roll dice (e.g., +roll strength + brawl)
|y+rouse|n             - Make a Rouse check
|y+feed|n              - Feed to reduce Hunger (after RP)
|y+stats|n             - Quick view of key statistics

|WCOMMUNICATION:|n

|y+bbs|n               - Access bulletin boards
|y+bbs/read <#>|n      - Read a BBS post
|y+bbs/post <board>=<subject>/<text>|n - Post to a board
|ychannel <text>|n     - Speak on a communication channel
|y+page <who>=<msg>|n  - Send private message

|WPOLITICAL SYSTEMS:|n

|y+status|n            - View Status hierarchy
|y+status/view <who>|n - Check someone's Status
|y+boons|n             - View boons owed/held
|y+boons/offer <who>=<type>/<desc>|n - Offer a boon

|WCHARACTER DEVELOPMENT:|n

|y+chargen|n           - Begin character creation
|y+xp|n                - View experience points
|y+spend <stat>|n      - Spend XP to improve stats
|y+bg <background>|n   - View background details

|WDISCIPLINE POWERS:|n

|y+use <discipline>|n  - Activate a discipline power
|y+powers|n            - List available discipline powers
|y+powers <discipline>|n - View specific discipline details

|WSTAFF COMMANDS:|n

|y@create|n            - Create objects (Builder+)
|y@dig|n               - Create rooms (Builder+)
|y@set|n               - Modify object properties (Builder+)
|y+job|n               - View/manage tickets (Staff)
|y+approve|n           - Approve characters (Staff)

|WCOMBAT & CONFLICT:|n

|y+init|n              - Join initiative tracker
|y+attack <target>|n   - Make an attack roll
|y+defend|n            - Make a defense roll
|y+damage <amount>|n   - Apply damage (after resolution)

|WMUSH ETIQUETTE:|n

- Commands provide |ymechanical support|n - you narrate the story
- Always |yrolep lay|n your actions, don't just use commands
- Use |y+roll|n for contested actions between players
- Coordinate with others for dramatic scenes
- Ask staff via |y+job|n if you need help
